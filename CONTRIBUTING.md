# AI-Generated Content Policy

This project does not accept content created with generative artificial intelligence (GenAI) or large language models (LLMs). All contributions created entirely or substantially with GenAI or LLMs will be removed. This includes code, documentation, and asset contributions, as well as issue and pull request descriptions.

GenAI may only be used for the purpose of understanding the codebase and its documentation or for user-reviewed debugging assistance, provided that the results are thoroughly reviewed and any suggested code is manually implemented.
